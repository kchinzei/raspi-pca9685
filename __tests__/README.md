Testing TypeScript Module
=========================

This directory
contains [ts-ject](https://github.com/kulshekhar/ts-jest) files.

### Prerequisites

Apply following commands in the root directory of the project.
```Shell
npm i -D jest typescript
npm i -D ts-jest @types/jest
```

### Running tests
```Shell
npx jest
```
